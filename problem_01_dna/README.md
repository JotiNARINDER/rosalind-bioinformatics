# Problem 01 — Counting DNA Nucleotides

**Source** : [Rosalind - Counting DNA Nucleotides](https://rosalind.info/problems/dna/)

## Énoncé
Étant donné une chaîne ADN de longueur maximale 1000 nt, compter le nombre 
d'occurrences de chaque nucléotide (A, C, G, T).

## Approche
Utilisation de la méthode `.count()` de Python pour compter chaque nucléotide 
dans la séquence.

## Résultats
* **Sample Output** (exemple Rosalind) : `20 12 17 21`
* **Dataset Output** (résultat final) : `210 235 198 213`
